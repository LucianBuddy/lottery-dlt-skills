# lottery-dlt-skills
A collection of skills for a series of Super Lotto prediction rules, intended for studying and learning lottery data‑driven algorithms.

DLT 大乐透多策略融合预测系统
六大章节：
核心架构 — 六池加权采样（6池×权重）、跨期模式识别（9种特征）、神经网络集成（TabNet/LSTM/Transformer 25%/35%/40%）、策略融合引擎（5组×5维度）、后处理管线（18步）
功能 — 3种投注类型（单式/复式12种/胆拖）、凯利公式资金管理、4项预测质量优化
快速开始 — Python API 示例 + CLI 命令（6个子命令）
数据 — 双源自动同步（体彩API + 500彩票网），~2,880期历史数据
文件结构 — 28个代码模块，~7,600行
版本历史 — V1.0 → V3.0.0 演进
